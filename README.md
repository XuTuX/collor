# 컬러 퍼즐 7

Love2D로 만든 색 조합 퍼즐 로그라이트입니다. 손패의 색친구를 실행 줄에 보내고, 모노/대칭/쌍둥이/크레센도/지그재그 규칙과 증강체 효과로 목표 점수를 넘기는 게임입니다.

## 실행

Love2D 11.4 이상이 필요합니다.

```sh
love .
```

## 조작

- 마우스 클릭: 색친구 선택
- 드래그: 손패 순서 바꾸기
- 실행 버튼 또는 `Enter`/`Space`: 선택한 색친구 실행
- 바꾸기 버튼 또는 `D`: 선택한 색친구 교체
- 주머니 또는 `B`: 남은 카드 보기
- `R`: 새 모험 시작
- `Esc`: 오버레이 닫기 또는 종료

## 핵심 구조

- `src/core`: 게임 상태, 저장, 상태 머신
- `src/data`: 캐릭터, 족보, 밸런스, 조커, 상점 아이템 데이터
- `src/gameplay`: 덱/손패/족보/턴 규칙
- `src/systems`: 점수, 조커, 애니메이션, 오디오, 이펙트
- `src/states`: 메뉴, 플레이, 결과, 상점 화면
- `src/ui`: 공통 UI 컴포넌트

상점 판매 풀은 `src/data/shop_items.lua`가 단일 소스입니다. 조커 수치와 설명은 `src/data/jokers.lua`에 두고, 상점은 그 데이터를 가져와 판매 카드로 변환합니다.

## 상점 아이템

상점에는 세 종류가 나옵니다.

- 증강체: 보유 슬롯에 장착되어 점수 계산 때 발동합니다.
- 반짝임: 특정 족보의 레벨을 올립니다.
- 아이템: 즉시 사용되어 영구 주머니를 바꿉니다. 색친구 추가, 삭제, 특정 색 삭제, 왼쪽 색을 오른쪽 색으로 변환하는 아이템이 있습니다.

상점 화면은 아이템/반짝임 구매 칸과 증강체 구매 칸을 분리해서 보여줍니다. 매 상점마다 아이템/반짝임 후보 2개와 증강체 후보 1개가 따로 뽑힙니다.

시간을 기다리면 이득을 주는 시간 기반 증강체는 제거했습니다. 플레이를 멈추고 기다리는 전략이 최적이 되지 않게 하기 위한 결정입니다.

## 저장

`src/core/save_manager.lua`는 Love2D 저장 공간에 `collor_save.lua`를 씁니다. 현재 저장 대상은 진행도, 골드, 보스 기믹, 조커, 영구 주머니, 족보 강화 상태입니다.
